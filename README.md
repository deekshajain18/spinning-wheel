# Spin Wheel App

Spin Wheel App is a web application designed to gamify user engagement through interactive spinning wheels.

## Features

- Spin Wheel: Users can spin a wheel to win prizes or rewards.
- Customizable Settings: Admin panel allows customization of wheel options and prizes.
- Admin Panel: Manage tournaments, users, and finances with ease.
- Responsive Design: Works seamlessly across devices for optimal user experience.

## Technologies Used

- HTML, CSS, JavaScript for frontend development.

## Installation

1. Clone the repository: `git clone https://github.com/your-username/spin-wheel-app.git`
2. Navigate to the project directory: `cd spin-wheel-app`
3. Install dependencies: `npm install`
4. Start the server: `npm start`
5. Open the app in your browser: `http://localhost:3000`

## Usage

- User Interface: Access the Spin Wheel interface from the homepage.
- Admin Panel: Access the admin panel by navigating to `/admin`.
- Customize Settings: Modify wheel options and prizes from the admin panel.
